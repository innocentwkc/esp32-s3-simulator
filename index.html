Why use the simulator

Everything you need to
prototype faster

From circuit design to running code — all in one place, in your browser.

🔌

Visual Circuit Builder

Drag components onto the canvas, click pins to draw wires, and build complete circuits visually. The ESP32-S3 board is fully rendered with all 40+ pins labeled and interactive.

⌨️

Arduino-Compatible Code

Write real Arduino code — pinMode, digitalWrite, delay, and Serial output all work exactly as on hardware.

⚡

Live Simulation

Hit Run and watch LEDs glow, buttons trigger events, and Serial output stream in real time. Control simulation speed from 0.25× to 10×.

📟

Serial Monitor

Serial.println() output appears instantly in the built-in monitor — just like the Arduino IDE, without a USB cable.

🎛️

Interactive Components

Press buttons while the simulation runs, spin the potentiometer with your scroll wheel — components respond in real time.

🌐

Zero Setup

No install, no drivers, no IDE. Open your browser and start building. Works offline after first load. Share circuits by saving the URL.

Workflow

From idea to simulation
in minutes

1

Place Components

Click a component in the palette, then click the canvas to place it. Drag to reposition. Right-click to delete.

2

Connect Wires

Click a pin on the ESP32-S3 board or a component pin to start a wire. Click a second pin to complete the connection.

3

Write & Run Code

Write Arduino code in the editor on the right. Hit Run — your code executes and components respond immediately.
